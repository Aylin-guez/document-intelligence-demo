# Before / After

## Input

`sample_data.json`

This file contains fictitious report data:

- report title
- company name
- summary points
- findings table rows
- timeline events
- conclusions
- appendix notes

## Output

`output/business_report_demo.pdf`

The output is a professional six-page PDF report built from that data.

## Simple explanation

The automation takes structured data and turns it into a clean report layout.
Instead of manually formatting each page, the same input can be used to generate a polished PDF consistently.

## Commercial angle

This is useful for clients who need repeatable reporting from data sources and want a document that looks ready for stakeholders.
