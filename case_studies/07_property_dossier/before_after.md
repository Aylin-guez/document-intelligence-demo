# Before / After

## Input

`sample_data.json`

Contains fictitious property data:

- property metadata
- location
- price
- area
- bedrooms and bathrooms
- features
- amenities
- pricing notes
- contact details
- next steps

## Output

`output/property_dossier_demo.pdf`

A seven-page professional real estate dossier PDF.

## Simple explanation

The automation takes structured property listing information and turns it into a clean PDF dossier.

Instead of manually formatting property summaries, features, amenities, pricing, and contact sections, the same structured input can generate a consistent client-ready document.

## Commercial angle

Useful for real estate agents, property managers, rental teams, investors, and agencies that repeatedly prepare property brochures or listing documents.
