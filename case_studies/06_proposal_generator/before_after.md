# Before / After

## Input

`sample_data.json`

Contains fictitious proposal data:

- client information
- project title
- proposal summary
- objectives
- scope of work
- deliverables
- timeline
- pricing
- terms
- next steps

## Output

`output/proposal_generator_demo.pdf`

A seven-page professional commercial proposal PDF.

## Simple explanation

The automation takes structured proposal information and turns it into a clean PDF document.

Instead of manually formatting proposal summaries, scope, deliverables, timelines, pricing tables, terms, and next steps, the same structured input can generate a consistent client-ready proposal.

## Commercial angle

Useful for freelancers, consultants, agencies, service providers, and teams that repeatedly prepare proposals or quotes.
