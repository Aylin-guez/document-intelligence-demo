# Before / After

## Input

`sample_data.json`

Contains fictitious brand book data:

- brand metadata
- brand overview
- values
- color palette
- typography
- voice and tone
- usage guidelines
- appendix notes

## Output

`output/brand_book_demo.pdf`

An eight-page professional brand book PDF.

## Simple explanation

The automation takes structured brand identity information and turns it into a clean PDF guide.

Instead of manually formatting values, colors, typography, voice rules, usage guidelines, and appendix sections, the same structured input can generate consistent brand documentation.

## Commercial angle

Useful for designers, agencies, startups, marketing teams, and businesses that repeatedly prepare brand books or style guides.
