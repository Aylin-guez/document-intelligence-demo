# Before / After

## Input

`sample_data.json`

Contains fictitious traceability data:

- source files
- matrix records
- matched references
- missing references
- exceptions
- recommendations
- appendix notes

## Output

`output/traceability_matrix_demo.pdf`

A seven-page professional traceability matrix report.

## Simple explanation

The automation takes source records and reference checks, then turns them into a clean PDF report with summary, matrix, exceptions, recommendations, and appendix.

## Commercial angle

Useful for audit, compliance, ISO, quality control, reconciliation, and evidence tracking workflows.
