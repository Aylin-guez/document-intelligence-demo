# Before / After

## Input

`sample_data.json`

This file contains fictitious internal audit data:

- findings
- risk levels
- affected areas
- evidence references
- recommendations
- statuses
- responsible owners
- next steps

## Output

`output/audit_report_demo.pdf`

The output is a seven-page professional internal audit report built from that structured data.

## Simple explanation

The automation takes audit findings and turns them into a polished report layout.
Instead of manually formatting a cover, summary, register, risk view, recommendations, evidence list, and conclusion, the same structured input can generate a consistent PDF.

## Commercial angle

This is useful for clients who need repeatable audit, compliance, quality, ISO, or risk documentation and want a deliverable that looks ready for review.
