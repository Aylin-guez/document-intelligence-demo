# Before / After

## Input

`sample_data.json`

Contains fictitious funding proposal data:

- organization profile
- project summary
- problem statement
- objectives
- activities and timeline
- budget
- impact metrics
- sustainability notes
- appendix notes

## Output

`output/grant_funding_proposal_demo.pdf`

An eight-page professional grant funding proposal PDF.

## Simple explanation

The automation takes structured project funding information and turns it into a clean proposal document.

Instead of manually formatting summaries, activities, budgets, impact metrics, and sustainability sections, the same structured input can generate consistent funding proposals.

## Commercial angle

Useful for NGOs, foundations, schools, municipalities, cultural projects, community programs, and teams that repeatedly prepare funding applications.
