# Before / After

## Input

`sample_data.json`

This file contains fictitious investigation data:

- case metadata
- people involved
- event timeline
- evidence records
- key findings
- recommendations
- next steps

## Output

`output/investigation_dossier_demo.pdf`

The output is a seven-page professional investigation dossier built from structured data.

## Simple explanation

The automation takes investigation records and turns them into a polished PDF layout.

Instead of manually formatting people tables, timelines, evidence registers, findings, and conclusions, the same structured input can generate a consistent dossier.

## Commercial angle

This is useful for clients who need repeatable investigation, compliance, HR, due diligence, or evidence documentation.
